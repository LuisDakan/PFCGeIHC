## Acerca del proyecto

El proyecto es un entorno virtual interactivo que representa una zona abierta como un parque o museo, e incluye elementos de la cultura prehispánica y la lucha libre.

El estilo visual está inspirado en diferentes universos ficticios:
- Ace Attorney
- Crash Bandicoot
- Juegos de SEGA
